# IFBA-NODEJS

Repositório organizado por disciplinas e assuntos, consolidando projetos e atividades desenvolvidos no curso de **Desenvolvimento de Software Back-End em Javascript** do IFBA.

## 📂 Estrutura do Repositório

A organização segue a matriz curricular e os temas abordados durante o curso:

### 01. Lógica de Programação
Exercícios e atividades focadas nos fundamentos da programação:
- Laços de Repetição (Loops)
- Manipulação de Arrays
- Condicionais e Operadores
- Algoritmos de ordenação e busca simples

### 02. Programação Orientada a Objetos (POO)
Projetos que aplicam os pilares da POO (Abstração, Encapsulamento, Herança e Polimorfismo):
- **AgendaC**: Sistema de gestão de contatos utilizando classes e métodos.

### 03. NodeJS e Express
Desenvolvimento de aplicações server-side e APIs:
- **Biblioteca-em-NodeJS**: Sistema completo de gerenciamento de biblioteca com integração de banco de dados e rotas.

### 04. Banco de Dados
Espaço reservado para scripts SQL, modelagens e integrações com bancos de dados.

### 05. Certificações e Documentos
Documentação oficial, certificados e materiais de apoio do curso.
