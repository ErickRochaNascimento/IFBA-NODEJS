# Resumo do Curso: Desenvolvimento de Software Back-End em Javascript

Este documento apresenta um resumo detalhado das competências e conteúdos abordados durante o curso de extensão realizado no **Instituto Federal da Bahia (IFBA)**, Campus Salvador, em parceria com o **CEPEDI**.

## 📋 Informações Gerais
- **Instituição:** IFBA - Campus Salvador
- **Período:** 02/09/2025 a 04/03/2026
- **Carga Horária Total:** 144 horas
- **Coordenação:** Prof. Dr. Joacir Simões Ferreira

## 📚 Matriz Curricular e Competências

### 1. Lógica de Programação com JavaScript (30h)
Desenvolvimento do raciocínio lógico aplicado à programação, utilizando a linguagem JavaScript para resolver problemas complexos através de algoritmos, estruturas de repetição e manipulação de dados.

### 2. Programação e Orientação a Objetos (18h)
Estudo dos pilares da POO: Abstração, Encapsulamento, Herança e Polimorfismo. Aplicação prática em sistemas modulares e escaláveis.

### 3. Node.js + Express (24h)
Construção de aplicações server-side de alta performance. Utilização do ecossistema Node.js e do framework Express para criação de servidores e gerenciamento de requisições.

### 4. Introdução a Web Services (18h)
Conceitos de arquitetura de serviços web, comunicação entre sistemas e integração de APIs.

### 5. Noções de Banco de Dados (18h)
Fundamentos de modelagem de dados, persistência de informações e integração de aplicações com sistemas de gerenciamento de banco de dados.

### 6. Padrões de Desenvolvimento de Software (18h)
Aplicação de boas práticas e padrões de projeto (Design Patterns) para garantir a qualidade, manutenção e escalabilidade do código.

### 7. Git e Versionamento (09h)
Domínio de ferramentas de controle de versão para trabalho colaborativo e gestão de histórico de código.

### 8. Soft Skills (09h)
Desenvolvimento de competências interpessoais, comunicação assertiva e trabalho em equipe no ambiente de desenvolvimento de software.
